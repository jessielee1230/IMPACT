WHAT IS A WEB SAFE FONT?

"Web Safe" fonts mean that they will be extremely common on most versions of Windows, Mac, Linux etc, so they will be viewable by more or less everyone. Along with each font is a preview image of what it looks like, and the raw CSS code that you can copy and paste directly into your own stylesheet to use the font.


I am not just a meme font but a stunning web safe font that you can use with CSS stylesheets. There is not just a single way to design me as you like.
